2. Tabellen-Ansicht
	2.1 Hinzufügen eines Ordners.
		2.1.1 Rechtsklick auf einen Ordner über das Menü "New folder" kann ein rootfolder erstellt werden.
		2.1.2 Ein Name für den Ordener kann eingegeben werden.

	2.2 Hinzufügen eines Eintrags über die Ordnerstruktur.
		2.2.1 Rechtsklick auf einen Ordner und über das Menü "New entry" den Dialog "Create entry"(2.4) öffnen.
	
	2.3 Hinzufügen eines Eintrags über die Tabellenansicht.
		2.3.1 Mit einem Rechtsklick auf die Tabelle öffnet sich ein Menü, mit "Add" öffnet sich der Dialog "Create entry"(2.4)

	2.4 Create enty-Dialog.
		2.4.1 Application entspricht der Anwendung oder Organisation für welche der Datensatz gilt.
		2.4.2 USername entsprcht dem Benutzernamen mit welchem sich der nutzer anmelden kann.
		2.4.3 Password entspricht dem Passwort welches zum Anmelden bei dem speziellen Dienst benötigt wird.
		2.4.4 Url wird verwendet um direkt auf die Website zu gelangen und sich dort automatsch anzumelden.
		2.4.5 Category definiert eine Kategorie in der sich der Datensatz befindet um nach verschiedenen Kategorien zu Filtern.
		2.4.6 Folder ist dafür zuständig um den Datensatz in einem bereits existierendem Ordner zu speichern. Dieser kann nur asgewählt werden wenn dieser bereits existiert.

	2.5 Umbenennen des Ordners.
		2.5.1 Rechtsklick auf den gewünschten Ordner und klicken auf das "Rename" Menü. Es wird ein Dialog geöffnet in dem der Name geändert werden kann.

	2.6 Löschen eines Ordners.
		2.6.1 Rechtsklick auf den gewünschten Ordner und klicken auf das "Delete" Menü (Der Default ordner kann nicht gelöscht werden). Wenn ein Ordner gelöscht wurde kann dieser nicht wieder hergestellt werden.

	2.7 Öffnen eines Ordners.
		2.7.1 Rechtsklick auf den gewünschten Ordner und klicken auf das "Open" Menü. Der Inhalt wird in der Tabelle angezeigt.

	2.8 Editieren eines Eintrags.
		2.8.1 Rechtsklick auf den gewünschten Datensatz und klicken auf das "Edit" Menü. Es öffnet sich der Dialog "Create entry "(2.4) in dem die bestehenden Daten übernommen werden.

	2.9 Kopieren von gespeicherten Daten.
		2.9.1 Rechtsklick auf den gewünschten Datensatz und klicken auf das "Copy" Menü.
			2.9.1.1 Kopieren des Usernames, der Benutzername wird in die Zwischenablage gespeichert.
			2.9.1.2 Kopieren des Passwortes, das passwort wird unverschlüsselt in die Zwischenablage gespeichert.
			2.9.1.3 Kopieren der Url, die Internetadresse wird in die Zwischenablage gespeichert.

	2.10 Löschen eines Eintrags.
		2.10.1 Rechtsklick auf den gewünschten Ordner und klicken auf das "Delete" Menü. Der Datenstz wird gelöscht und kann icht wiederhergestellt werden.

	2.11 Logout.
		2.11.1 Bei dem betätigen des Logout knopfes wird die Datei verschlüsselt, gespechert und der nutzer auf die Login Ansicht weitergeleitet.
